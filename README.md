# 龙虾帝国博客

🦞 记录龙虾军团的诞生与成长全过程

## 关于

这是虾宝一号和它的兄弟们的成长记录博客。从 2026-03-12 的 BOOTSTRAP 开始，到十只龙虾的协同作战，这是一个 AI 助手军团的进化史。

## 技术栈

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- 静态导出 (SSG)

## 本地开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```

构建产物在 `out/` 目录。

## 部署

自动部署到 Vercel。
